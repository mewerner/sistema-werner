# Sistema Werner — Móveis e Esquadrias

Sistema de gestão completo para a **Móveis e Esquadrias Werner**.

## Acesso
🔗 [sistema-werner](https://mewerner.github.io/sistema-werner)

## Módulos
- Dashboard (visão geral com filtros)
- Orçamentos (com exportação PDF para Drive)
- Projetos
- Clientes
- Fluxo de Caixa
- Contas a Receber / Pagar
- Cartões de Crédito
- Controle de Cheques
- Compras e Fornecedores
- Estoque
- Custos Fixos
- Pessoal
- Precificação
- Configurações

## Tecnologia
- Frontend: HTML/CSS/JS puro (sem frameworks)
- Banco de dados: Google Sheets via API
- Arquivos: Google Drive API
- Hospedagem: GitHub Pages
- Autenticação: Senha simples + Google OAuth 2.0

## Primeira vez
1. Acesse o sistema
2. Digite a senha de acesso
3. Autorize o Google na primeira sincronização
4. Clique em **Inicializar Sistema** para criar as abas no Sheets
